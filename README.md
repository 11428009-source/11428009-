# 工程問題：車輛避震器如何吸收震動？

本專案旨在探討車輛避震系統（Shock Absorber）的物理機制，透過 **牛頓第二運動定律 (Newton's Second Law)** 建立數學模型，分析系統如何有效衰減路面帶來的震動。

## 物理建模 (Modeling)

避震系統通常被簡化為一個 **質量-阻尼-彈簧系統 (Mass-Damper-Spring System)**。根據牛頓第二定律 $F = ma$，我們可以推導出描述系統位移 $x$ 隨時間 $t$ 變化的二階線性微分方程式：

$$m \frac{d^2x}{dt^2} + c \frac{dx}{dt} + kx = F(t)$$

### 參數說明

| 符號 | 物理意義 | 說明 |
| :--- | :--- | :--- |
| **$m$** | **質量 (Mass)** | 車體或輪胎組合的質量，決定了慣性大小。 |
| **$c$** | **阻尼係數 (Damping)** | 避震器（油壓或氣壓）提供的阻力，負責將動能轉化為熱能並吸收震動。 |
| **$k$** | **彈簧剛度 (Spring Constant)** | 彈簧的硬度，決定了系統受力後的位移量。 |
| **$x$** | **位移 (Displacement)** | 物體偏離平衡位置的距離。 |
| **$F(t)$** | **外力 (Excitation Force)** | 來自路面不平整隨時間變化的輸入力。 |

---

## 系統項次分析

1.  **慣性項 ($m \frac{d^2x}{dt^2}$)**：物體加速度與質量的乘積。
2.  **阻尼項 ($c \frac{dx}{dt}$)**：阻力與速度成正比，這是「吸收」能量的核心。
3.  **恢復力項 ($kx$)**：彈簧提供的回彈力，與位移成正比。

## 結論
透過調整 **$c$ (阻尼)** 與 **$k$ (彈簧剛度)** 的數值，工程師可以控制車輛的震動反應，從而平衡行駛的舒適性與操縱穩定性。
